# 문서 목록 (Documentation Index)

본 디렉토리는 `Next.js + TipTap 기반 Markdown 에디터` 프로젝트의 기술 및 기능 문서를 포함합니다.

## 공통/기획 문서
*   **[GRD.md](./GRD.md)**: 기능 요구서 (General Requirements Document). 프로젝트의 목적과 핵심 기능을 정의합니다.
*   **[GUI.md](./GUI.md)**: GUI 명세서. 화면 구성, 레이아웃, 인터랙션 및 상태 정의를 다룹니다.
*   **[GUI.md](./GUI.md)**: GUI 명세서. 화면 구성, 레이아웃, 인터랙션 및 상태 정의를 다룹니다.
*   **[GUI_FLOW.md](./GUI_FLOW.md)**: GUI 인터랙션 흐름. 사용자 행동에 따른 상태 변화와 데이터 흐름을 시각화합니다.
*   **[GUI_WIREFRAME.md](./GUI_WIREFRAME.md)**: GUI 와이어프레임. 기능별 화면 구성을 텍스트(ASCII)로 시각화했습니다.
*   **[DESIGN_GUIDE.md](./DESIGN_GUIDE.md)**: 디자인 가이드. 컬러, 타이포그래피, 컴포넌트 스타일 등 시각적 정의를 포함합니다.
*   **[SEQ_DIAGRAM.md](./SEQ_DIAGRAM.md)**: 주요 기능(이미지 업로드, 붙여넣기 등)의 시퀀스 다이어그램입니다.

## 기술/설계 문서
*   **[TRD.md](./TRD.md)**: 기술 요구서 (Technical Requirements Document). 시스템 아키텍처, 데이터 흐름, 기술 스택 등을 정의합니다.
*   **[API.md](./API.md)**: 서버 API 명세서. 이미지 업로드 API 등의 상세 스펙을 다룹니다.
*   **[PROJECT_STRUCTURE.md](./PROJECT_STRUCTURE.md)**: 프로젝트 폴더 구조 및 파일 구성 안내입니다.
*   **[DEVELOPMENT.md](./DEVELOPMENT.md)**: 개발 환경 설정, 빌드, 배포 가이드입니다.

## 테스트/검증 문서
*   **[FUNCTIONS.md](./FUNCTIONS.md)**: 기능 목록 및 테스트 케이스 매핑표입니다.
*   **[TEST_CASE.md](./TEST_CASE.md)**: 상세 테스트 케이스 정의서입니다.
