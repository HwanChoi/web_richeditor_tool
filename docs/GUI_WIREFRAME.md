# GUI 와이어프레임 (Text Mockup)

본 문서는 GUI의 배치와 기능별 시각적 구성을 텍스트(ASCII Art) 형태로 표현합니다.
실제 구현 시 화면 구성의 가이드라인으로 활용합니다.

## 1. 전체 레이아웃 (Desktop Split View)

기본적인 2열 편집 화면 구성입니다.

```text
+-------------------------------------------------------------------------------+
|  Toolbar (Sticky)                                                             |
| +---------------------------------------------------------------------------+ |
| | [B] [I] [S] | [H1] [H2] [H3] | [UL] [OL] [Qt] | [HR] | [Code] | [Img]     | |
| +---------------------------------------------------------------------------+ |
+-------------------------------------------------------------------------------+
|                                       |                                       |
|  [ Rich Editor Area ]                 |  [ Markdown Source Area ]             |
|                                       |                                       |
|  (H1) 프로젝트 기획안                 |  # 프로젝트 기획안                    |
|                                       |                                       |
|  여기는 리치 텍스트 에디터입니다.     |  여기는 리치 텍스트 에디터입니다.     |
|  **굵은 글씨**와 *기울임*을           |  **굵은 글씨**와 *기울임*을           |
|  바로 확인할 수 있습니다.             |  바로 확인할 수 있습니다.             |
|                                       |                                       |
|  - 기능 1                             |  - 기능 1                             |
|  - 기능 2                             |  - 기능 2                             |
|                                       |                                       |
|  +-------------------------+          |  ![](/uploads/image_1.png)            |
|  | [ Image Preview ]       |          |                                       |
|  |                         |          |                                       |
|  +-------------------------+          |                                       |
|                                       |                                       |
|                                       |                                       |
|                                       | +-----------------------------------+ |
|                                       | | [ < Markdown -> 에디터 반영 > ]   | |
|                                       | +-----------------------------------+ |
+-------------------------------------------------------------------------------+
| [ .md 파일로 저장 (Download) ]                                                |
+-------------------------------------------------------------------------------+
```

## 2. 툴바 기능별 상세 (Toolbar Detail)

툴바의 버튼 그룹핑과 기능 아이콘 배치입니다.

```text
[ 텍스트 서식 ]   [ 문단 서식 ]           [ 편집 도구 ]    [ 특수 삽입 ]
+-------------+ +-----------------------+ +--------------+ +----------------+
| [B] [I] [~] | | [H1] [H2] [H3]        | | [Undo][Redo] | | [Code] [Image] |
+-------------+ +-----------------------+ +--------------+ +----------------+
 Bold  Italic     Heading 1/2/3           History Control   CodeBlock
       Strike                                               Image Upload
```

## 3. 기능별 UI 상태 (Functional States)

### 3.1 이미지 업로드 진행 중 (Image Uploading)

사용자가 이미지를 선택하거나 드롭했을 때의 피드백입니다.

```text
   Editor Area
+-----------------------------------------------------+
|                                                     |
|  문서를 작성 중입니다...                            |
|                                                     |
|  +-----------------------------------------------+  |
|  |  [ Spinner ] 이미지 업로드 중... (45%)        |  |
|  +-----------------------------------------------+  |
|                                                     |
+-----------------------------------------------------+
```

### 3.2 코드블럭 편집 (Code Block Editing)

코드블럭 삽입 시의 신택스 하이라이팅 영역 표현입니다.

```text
   Editor Area
+-----------------------------------------------------+
|  다음은 예제 코드입니다.                            |
|                                                     |
|  +-----------------------------------------------+  |
|  | python                          [Copy] [Del]  |  | <--- Header (Optional)
|  |-----------------------------------------------|  |
|  | 1  def hello_world():                         |  |
|  | 2      print("Hello")                         |  |
|  +-----------------------------------------------+  |
|                                                     |
+-----------------------------------------------------+
```

### 3.3 모바일 레이아웃 (Mobile Stack View)

좁은 화면에서의 수직 배치입니다.

```text
+---------------------------------------+
| [Toolbar] [B] [I] [list] ... [menu]   |  <-- 가로 스크롤 가능
+---------------------------------------+
|                                       |
|  [ Rich Editor ]                      |
|  Min-Height: 300px                    |
|                                       |
+---------------------------------------+
|  ---------------- (Divider) --------- |
+---------------------------------------+
|                                       |
|  [ Markdown Source ]                  |
|  Min-Height: 200px                    |
|                                       |
+---------------------------------------+
| [ Apply Button ]                      |
| [ Download Button ]                   |
+---------------------------------------+
```
